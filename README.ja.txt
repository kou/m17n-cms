= README

== 名前

M17N CMS

== これはなに？

M17N CMSは静的なHTMLのサイト用のコンテンツ管理システムです。
（つまり、CGIを使わないサイト）M17N CMSは以下の機能を提供しま
す。

* 静的なHTML生成機能。
* 静的なHTMLをFTPサーバにアップロードする機能。
* コンテンツをWYSIWYGで編集できるインターフェイス。
